This application provides support for conference management. It allows to organize conferences alongside their most important components:

  • miscellaneous details about each upcoming and ongoing conference
  • uploading papers
  • a system for reviewing the papers
  • how the conference will be split into sections and what will be presented there
  • the ability for each user to choose which presentation(s) he/she will attend

In order to access most of the functionalities of this application you must first register.

After you have an account, the Chair and CoChairs can give you certain credentials that will enable you to use specific functionalities of the system.
